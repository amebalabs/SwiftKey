About SwiftKey Snippets

SwiftKey Snippets is a community-driven repository of configuration snippets for SwiftKey. Snippets are small, reusable pieces of YAML configuration that can be easily imported into your SwiftKey setup.

Share Your Snippets

Have a useful configuration that others might benefit from? Share it with the community by contributing to the SwiftKey Snippets repository.

1

Create a Snippet

Create a YAML configuration snippet that includes a useful set of menu items. Make sure to include a descriptive name, tags, and a clear explanation of what your snippet does.

2

Fork the Repository

Fork the SwiftKey Snippets repository on GitHub.

3

Add Your Snippet

Add your snippet to the repository and create a pull request. Your snippet will be reviewed and merged if it meets the quality standards.
